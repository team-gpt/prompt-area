# prompt-area

A [shadcn](https://ui.shadcn.com/) registry component for AI prompt input.

## Usage

Install via shadcn CLI:

```bash
npx shadcn@latest add https://<your-domain>/r/prompt-area.json
```

## Development

```bash
pnpm install
pnpm dev
```

## Build Registry

```bash
pnpm registry:build
```
